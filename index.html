How To Play:

  • Match up tile with the same number. The tile will be incremented by one
  • You get a new row when you run out of move OR you run out of time
  • Score is equal to highest number in the field
  • If you get you win

Select Difficulty

  • - Relatively high number will appear
  • - Less ties will be created
  • - More time available before new row is created
  • - Moderate amount of ties will be created
  • - Normal time interval before new row is created
  • - Higher amount of ties will be created
  • - Less chance of getting high number when the new row is created
  • - Ties will be created all over the place
  • - Less time available before new row is created
